
# 75 Hard Premium PWA

## Deploy to Vercel or Netlify
1. Upload all files in this folder to a new GitHub repo.
2. Import the repo into Vercel or Netlify.
3. Build command: `npm run build`
4. Output directory: `dist`

## Install on phone
Open the live URL in Chrome (Android) or Safari (iPhone) and install/add to home screen.
